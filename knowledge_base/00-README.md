# Base de conocimiento — Grupo Sazón (fuente para RAG)

Documentos de referencia que el asistente de selección usa para responder
preguntas de los candidatos (FAQ) durante el cribado. Están pensados para
trocearse (*chunking*) y vectorizarse.

- **Idioma:** español (España y México). El asistente traduce al inglés cuando el
  candidato escribe en inglés.
- **Tono:** claro, cercano y honesto. Para cifras concretas (salario exacto, fecha
  de incorporación, detalles del contrato) el asistente remite siempre al
  reclutador; estos documentos solo dan información general.
- **Aviso:** Grupo Sazón es una empresa ficticia. Estos contenidos se han creado
  para la demo de la prueba técnica y no describen a ninguna compañía real.
